# Travel Destination Planner

Travel Destination Planner is a polished React + Vite portfolio project that helps users explore destinations through live weather, daylight, and trip planning data from Open-Meteo.

## Features

- Destination-first travel hero with live weather details
- Search suggestions powered by Open-Meteo geocoding
- Current weather, hourly forecast, and 7-day outlook
- Destination overview with timezone, coordinates, and local time
- Sunrise and sunset cards
- Best day to visit scoring
- Saved trips with localStorage persistence
- Recent searches and theme persistence
- Responsive light and dark UI
